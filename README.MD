# Webpack Starter

Este es el repositortio inicial para crear aplicaciones utilizando WebPack. Esto es una prueba de actualización. 

### Notas:

Recuerden reconstruir los módulos de Node

```
npm install
```

Y para construir el build, recuerden:

### Producción:
```
npm run build
```
### Desarrollo: 
```
npm run build:dev
```
